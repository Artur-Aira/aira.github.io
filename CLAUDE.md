\# AIRA Portfolio



\## Общая информация



Это персональное портфолио дизайнера Артура.



Основная задача сайта —

получать клиентов через современный премиальный дизайн.



\## Стиль



Минимализм.



Много воздуха.



Большая типографика.



Плавные анимации.



Без визуального шума.



Вдохновение:



Apple



Linear



Stripe



Framer



\## Правила



Не использовать Bootstrap.



Не использовать Tailwind.



Не использовать jQuery.



Использовать только:



HTML



CSS



JavaScript



Все изменения должны сохранять адаптивность.



Перед изменением сначала анализировать код.



После изменения проверять ошибки.



Не удалять существующие секции без необходимости.



Все новые секции должны соответствовать текущему стилю.



Писать чистый и понятный код.



Всегда предлагать улучшения UX.
